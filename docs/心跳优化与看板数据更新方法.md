# 台架心跳优化与看板数据更新方法

## 一、台架心跳优化

### 1.1 心跳机制说明

**优化后的心跳逻辑：**

| 参数 | 值 | 说明 |
|-----|---|------|
| 心跳间隔 | 30秒 | 每30秒发送一次心跳 |
| 离线判定 | 4次无响应 | 连续4次心跳无响应才判定离线 |
| 离线时间 | 约2分钟 | 4次 × 30秒 = 120秒 |

### 1.2 心跳流程

```
台架端                    看板前端                    后端
  │                          │                         │
  │──── 心跳请求(30s) ───────│────────────────────────>│
  │                          │                         │
  │<─── 心跳响应 ────────────│<────────────────────────│
  │                          │                         │
  │    ┌─ 收到响应 ──────────│── 更新状态为在线         │
  │    │                     │   更新测试项             │
  │    │                     │   重置失败计数=0         │
  │    │                     │                         │
  │    └─ 无响应 ────────────│── 失败计数+1             │
  │                          │   失败计数<4: 保持在线   │
  │                          │   失败计数>=4: 标记离线  │
```

### 1.3 前端心跳实现

```typescript
// frontend/src/hooks/useWebSocket.ts

const HEARTBEAT_INTERVAL = 30000; // 30秒
const OFFLINE_THRESHOLD = 4; // 4次无响应判定离线

const sendHeartbeat = useCallback(() => {
  if (wsRef.current?.readyState === WebSocket.OPEN) {
    missedHeartbeatsRef.current++;
    
    if (missedHeartbeatsRef.current >= OFFLINE_THRESHOLD) {
      // 标记所有台架离线
      benches.forEach(bench => {
        if (bench.status.state !== 'offline' && bench.status.state !== 'maintenance') {
          updateBenchStatus(bench.id, { state: 'offline' });
        }
      });
    } else {
      // 发送心跳ping
      wsRef.current.send(JSON.stringify({ event: 'ping', data: {} }));
    }
  }
}, [benches, updateBenchStatus]);
```

### 1.4 台架端心跳发送

**Python 示例：**

```python
import requests
import time

class BenchClient:
    def __init__(self, bench_id: str, server_url: str = "http://localhost:8000"):
        self.bench_id = bench_id
        self.heartbeat_url = f"{server_url}/api/v1/benches/{bench_id}/heartbeat"
        self.heartbeat_interval = 30  # 30秒
    
    def send_heartbeat(self, status: str = "running", current_task: str = None):
        """发送心跳"""
        payload = {
            "status": status,
            "current_task": current_task,
            "metrics": {}  # 可选：实时指标
        }
        
        try:
            response = requests.post(self.heartbeat_url, json=payload, timeout=5)
            return response.status_code == 200
        except:
            return False
    
    def run(self):
        """主循环"""
        while True:
            self.send_heartbeat(
                status="running",
                current_task="当前测试任务名称"
            )
            time.sleep(self.heartbeat_interval)

# 使用
client = BenchClient("your-bench-id")
client.run()
```

## 二、DVP进度看板

### 2.1 项目增加方法

**API 端点：** `POST /api/v1/dvp/projects`

**请求体：**
```json
{
  "name": "电机控制器V2.0",
  "description": "电机控制器DVP验证项目",
  "total_experiments": 20,
  "total_devices": 150
}
```

**响应：**
```json
{
  "project_id": "dvp_20260325120000_0",
  "name": "电机控制器V2.0",
  "description": "电机控制器DVP验证项目",
  "total_experiments": 20,
  "total_devices": 150,
  "progress": 0.0,
  "param_checked": false,
  "is_interrupted": false,
  "created_at": "2026-03-25T12:00:00",
  "updated_at": "2026-03-25T12:00:00"
}
```

**前端调用示例：**

```typescript
import { projectAPI } from '@/lib/dvp-api';

// 创建项目
const newProject = await projectAPI.create({
  name: "电机控制器V2.0",
  description: "电机控制器DVP验证项目",
  total_experiments: 20,
  total_devices: 150
});
```

### 2.2 进度更新方法

**API 端点：** `PUT /api/v1/dvp/projects/{project_id}/progress`

**请求体：**
```json
{
  "progress": 45.5,
  "completed_devices": 68,
  "param_checked": true,
  "is_interrupted": false
}
```

**前端调用示例：**

```typescript
import { projectAPI } from '@/lib/dvp-api';

// 更新进度
await projectAPI.updateProgress("dvp_001", {
  progress: 45.5,
  completed_devices: 68,
  param_checked: true
});

// 中断项目
await projectAPI.interrupt("dvp_001");

// 恢复项目
await projectAPI.resume("dvp_001");

// 标记参数已检查
await projectAPI.checkParams("dvp_001");
```

## 七、使用示例

### 7.1 自动化测试完整流程

```typescript
// 1. 创建项目
const project = await automationAPI.createProject({
  name: "电机控制器V3.0自动化测试",
  description: "MCU自动化测试项目"
});

// 2. 每日执行后添加记录
await automationAPI.addExecution({
  project_id: project.id,
  execution_date: "2026-03-25",
  test_cases: 200,
  execution_time_hours: 3.5,
  passed_count: 195,
  failed_count: 5
});

// 3. 查看统计数据
const metrics = await automationAPI.getMetrics('month');
console.log(`总用例: ${metrics.overview.total_test_cases}`);
console.log(`通过率: ${metrics.overview.pass_rate}%`);
```

```typescript
// 1. 获取活动类型
```
